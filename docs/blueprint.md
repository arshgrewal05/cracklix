# **App Name**: PrepStation

## Core Features:

- Centralized Exam Catalog: Browse and search a library of supported competitive exams stored in the database.
- Live Proctored Mocks: Participate in timed mock exam sessions with a dynamic question palette and real-time timer tracking.
- AI Rationalization Tool: An AI-powered tutor tool that provides logical reasoning and step-by-step explanations for complex mock questions.
- Admin Dashboard: Full CRUD interface for managing question banks, exam categories, and mock scheduling.
- Performance Insights: Detailed breakdown of results by topic and difficulty level stored securely in the firebase database.
- Responsive Exam Layout: Optimized interface for high-focus testing environments across desktop and tablet.

## Style Guidelines:

- Primary color: Electric Blue (#4785FF) to convey confidence and stability.
- Background color: Deep Charcoal (#16181D), a dark theme to reduce eye strain during prolonged testing sessions.
- Accent color: Vibrant Cyan (#17CCE5) for progression bars and active navigation states.
- Headline font: 'Space Grotesk' (Sans-serif) for its high-tech and precise technical feel; Body font: 'Inter' (Sans-serif) for maximum clarity and objective look.
- Outline-style, geometric icons for toolbars and question types to maintain a clean, administrative aesthetic.
- Fixed sidebar navigation for quick access to the Question Palette and Timer during active mock attempts.
- Subtle, non-distracting CSS transitions for the countdown timer and question transitions.
